## Error handling

```php
try {
    $response = $client->post('/shipments', ['json' => $data]);
} catch (ClientException $e) {
    printf('An client exception occurred: %d %s', $e->getCode(), TrackMageClient::error($e);
} catch (\Exception $e) {
    printf('An error occurred: %d %s', $e->getCode(), $e->getMessage());
}
```
